## Overview

A modern, responsive portfolio website showcasing my skills, projects, and professional experience as a Full Stack Developer. Built with performance and user experience in mind, featuring smooth animations and a clean, accessible design.

**🌐 Live Demo:** [portfolio-d26n.vercel.app](https://portfolio-d26n.vercel.app/)

## Key Features

### 🎨 Design & User Experience
- **Responsive Design** - Optimized for all devices and screen sizes
- **Dark/Light Theme** - Toggle between themes for better user preference
- **Smooth Animations** - Enhanced user interactions with Framer Motion
- **Accessibility First** - WCAG compliant design patterns

### ⚡ Performance & Development
- **Lightning Fast** - Vite-powered development and build process
- **Modern React** - Built with React 18+ and modern hooks
- **Modular Architecture** - Clean, maintainable component structure
- **SEO Optimized** - Meta tags and structured data for better visibility

### 🛠️ Functionality
- Interactive skills and tools showcase
- Project gallery with live demos and source code links
- Integrated resume/CV section
- Contact form with validation
- Smooth page transitions and micro-interactions

## Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/pran-ekaiva006/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
